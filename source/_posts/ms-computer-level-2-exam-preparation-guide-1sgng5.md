# MS计算机二级备考攻略

# 前言

虽然说计算机二级证书，对于就业其实用处不是很大，但是考这个证书更多的是时候是促使我们去学习办公软件的使用，MS OFFICE相对于WPS我觉得是比较难的，因为MS OFFICE有四十套题目，我觉得这个数量练完的话，恰好能够让你熟练掌握一些办公技巧，特别是EXCEL，这四十套题目里面有VLOOKUP,COUNTIFS,SUM,ROUND等等这些函数的运用。我先声明，我的备考是差不多两个月的时间，所以想要几天内速成的，我这里没有分享的技巧，当然这两个月的时间，我是三天打鱼两天筛网的状态。

# 准备工作

记得报名。然后可以准备一个笔记本，你可以记操作题的步骤，重点更多的是记忆选择题。没必要报班学习，没必要买书学习，这个钱拿去买杯奶茶，吃个麦当劳。

# PART1---操作题

操作题分为三大部分，EXCEL, WORD, PPT,其中最简单的是PPT,所以占的分也比较少，占20分。备考我建议先从EXCEL或者WORD入手，我一开始是每天做3套题目，EXCEL,WORD,PPT分别做一套，我不建议大家这样做，因为会影响你的熟练度，我后期发现，如果一直练一个专题，你操作的熟练度，以及你对界面的记忆会比较好。在练习的过程中，对于很少接触MS OFFICE的同学，你可以拿本子做一下笔记，记录它的操作顺序，然后自己独立的再做一遍题目，后面慢慢熟悉操作界面了，你就可以分屏操作，左边放视频，右边放题目，同时进行，这样刷题效率很高，我不建议边看手机，边写题目，首先自控力不高的，一般看着看着就刷视频去了，其次，分屏操作，你的操作界面变小了，所以你可以更加知道，哪个功能放在哪个区域，如图

![image](http://127.0.0.1:60234/assets/image-20260628152909-ssjlmof.png)

我建议是40套excel最要都要做一遍，因为很多excel题考查的内容是不一样的，有的数据透视图比较多，有的是计算函数比较多，侧重点不一样，word和ppt我都只写了20套，所以在考场上我的word做的不是很好，但是写20套word能够保证你的基础分，ppt的话20套基本考点基本都差不多，ppt你可以后面只挑一些难的做，word你可以挑不同考察内容的做，比如邮件合并类，修改论文格式，书目类等等。  
总而言之，我的总结是（专题训练）  
1一开始可以做笔记熟悉操作的流程，独立做一遍  
2熟悉操作界面后，分屏边看视频边做，偶尔可以记一些比较容易忘记的操作  
3后买如果时间紧张，可以挑不同类型和比较难的题目做  
4重要程度：excel>=word>ppt

# PART2---选择题

选择题也是看视频，看我在准备工作那里发的视频链接，你花一周视频就可以速通，3到4天差不多就足够，我的备考建议可以放到最后，前面尽量刷操作题，最后几天再看选择题。  
最后，希望大家都能够顺利通过考试，拿到证书。
